-------------------------------------
JSON-lib - JavaScript Object Notation
-------------------------------------

"JSON-lib is a java library for transforming beans, maps, collections, java 
arrays and XML to JSON and back again to beans and DynaBeans. It is based 
on the work by Douglas Crockford in http://www.json.org/java."

For more information:
http://json-lib.sourceforge.net/
http://www.json.org/
