------------------------
Apache Commons BeanUtils
------------------------

"The Java language provides Reflection and Introspection APIs (see the 
java.lang.reflect and java.beans packages in the JDK Javadocs). However, these 
APIs can be quite complex to understand and utilize. The BeanUtils component 
provides easy-to-use wrappers around these capabilities."

For more information:
http://commons.apache.org/beanutils/
