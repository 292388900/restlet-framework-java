Implementation of the JAX-RS 1.0 API developped by the JSR-311 expert group. The main class is JaxRsApplication.

This extension doesn't pretend to be fully compliant with the specifications but it is close to completion. We are looking forward to getting access to the TCK in order to validate our compatibility.
