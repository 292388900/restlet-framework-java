Implementation of the JAX-RS API based on the JSR-311 specification. The main class is JaxRsApplication.

This extension is currently under development. You should use this extension only for experimental purpose.
Send your comments to discuss@restlet.tigris.org. If you find a bug, write a bug report on the Restlet issue tracker - thanks.
