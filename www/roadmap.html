Roadmap


Goal

The Restlet project is first composed of the Restlet API which contains a set of interfaces and classes that developpers can use directly without being tied to a particular implementation. In addition, there is one implementation of this API, named the Noelios Restlet Engine (NRE), which is supported by Noelios Consulting. It also receives significant contributions from our community of users and developers.

Our goal is to release a standardized version 1.0 of the Restlet API and at the same time a complete and stable version of the NRE. Based on the smae code base, we plan to provide two complementary editions of the NRE, a Community Edition and a Standard Edition. Here are the details of each subproject:

  • Restlet API
    • open to multiple implementations and extensions
    • CDDL license which makes it freely available for all usages
    • standardization throught the Java Community Process (JCP)
  • NRE Community Edition
    • identical to the current NRE code base and distribution
    • CDDL license which makes it freely available for all usages
    • free support via our discussion list
  • NRE Standard Edition
    • same code base as the Community Edition
    • professional support from Noelios Consulting
    • guaranted response time
    • priority bug fixing
    • personal alerts for new versions and security issues
    • annual subscription fee

Conclusion

Due to the open source nature of the Restlet project, the timely completion of the following roadmap will depend on the actual time resources available. We will do our best to respect it and update it as frequently as possible. Please, feel free to contact us for questions and comments.
