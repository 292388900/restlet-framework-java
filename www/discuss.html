Discussion list

discuss@restlet.tigris.org

This mailing list dedicated to all discussions on the Restlet project.
In order to participate, you can use the following options:

  • You can read and search the archived messages on the Tigris web site.
  • With your email client, you can:
    • subscribe by sending an email to discuss-subscribe@restlet.tigris.org.
    • post a message by sending an email to discuss@restlet.tigris.org.
    • unsubscribe by sending an email to discuss-unsubscribe@restlet.tigris.org.
  • Using the Gmane service you can acces to the list via three types of interfaces:
    • NNTP/Newsgroup reader like Thunderbird or Outlook Express.
    • Web interface with frames and threads of discussion.
    • Web interface with blog-like, flat list of messages.
    • RSS/Feeds reader like Firefox or Thunderbird.
